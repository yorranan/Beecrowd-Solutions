# Soluções do Beecrowd

Repositório para inserir soluções do Beecrowd (Antigo URI) em C, C++ e Python.

### Como executar:

> Para códigos em C++ substitua o comando `gcc` por `g++`.

**C - Powershell:**

```shell
gcc 1000.c -o 1000
./1000
```

**C - CMD**
```cmd
gcc 1000.c -o 1000 && 1000
```

**C - Linux**

```bash
gcc 1000.c -o 1000 && ./1000
```

**Pyhton:**

```bash
python 100.py
```
